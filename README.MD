## 🧠 核心學習筆記

在刷題過程中，我整理了一些通用的技術重點：

### 1. Linked List (鏈結串列)
* **技巧**：Dummy Node (虛擬頭節點) 的運用，能有效簡化邊界條件。
* **體會**：理解「節點物件」與「數值」的差異是避免 `TypeError` 的關鍵。

### 2. Hash Table (哈希表)
* **技巧**：利用 `dict` 達到 $O(1)$ 的查詢速度。
* **體會**：「邊找邊放」的策略比「先放再找」更能優化空間並解決重複值問題。
